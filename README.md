# automating-aws-with-python

Repository for the A Cloud Guru course *Automating AWS with Python*

## 01-webotron

webotron is a script that will sync a local directory to an S3 bucke, and optionally configure Route53 and cloudfront as well

###Features

weobtron currently has the following features
- List buckets
- List bucket objects
- Create and set up bucket for website
- sync directory tree to bucket
